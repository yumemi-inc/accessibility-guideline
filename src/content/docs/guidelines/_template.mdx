### Bad: 画像にalt属性が無い

```html
<img src="image.png">
```

### Good: アイコンに適切な代替テキストが設定されている

sr-only (visually-hidden) を使用して代替テキストを提供する例:

```html
<svg width="100" height="100" aria-hidden="true">
 <!-- ... -->
</svg>
<span class="sr-only">アイコン：〜〜〜</span>
```

aria-label を使用して代替テキストを提供する例:

```html
<svg width="100" height="100" role="img" aria-label="アイコン：〜〜〜">
 <!-- ... -->
</svg>
```

### Bad: アイコンに代替テキストが設定されていない

```html
<svg width="100" height="100" aria-hidden="true">
```
